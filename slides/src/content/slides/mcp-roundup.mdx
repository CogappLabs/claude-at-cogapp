## Cogapp's MCP roundup

| MCP | Who uses | What for |
| :---- | :---- | :---- |
| Linear | Devs | Tickets, projects, docs |
| Slack | All | Search, post, summarise |
| Atlassian | Producers | Jira, Confluence |
| claude-in-chrome | Frontend | Browser test, console logs |
| Playwright | Frontend | E2E test automation |

**Browser tools — pick one:**

- **Playwright** — much faster (headless, parallel). Use for repeatable test flows.
- **claude-in-chrome** — drives *your real Chrome* with your sessions/cookies. Use when you need authenticated state, extensions, or real DevTools console.
