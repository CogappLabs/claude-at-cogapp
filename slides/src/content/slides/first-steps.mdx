## First steps

Good starter prompts:

- "Review my staged changes for bugs and edge cases"
- "Write a commit message for these changes"
- "Debug this failing test: [paste]"
- "What edge cases am I missing in this validator?"
- "Write tests for `formatDate` covering main flows"
- "Explain what this middleware does"

Avoid: rewriting whole modules, anything irreversible.
