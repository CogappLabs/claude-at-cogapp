## Subagents & parallel work

Main agent fans out:

Built-in:
- **Explore (Haiku)**: file lookup, grep, single-file Q&A
- **general-purpose**: open-ended research

From `feature-dev` plugin:
- **code-reviewer**: second opinion on diff
- **code-explorer**: trace execution paths
- **code-architect**: design new features

Tell Claude to fan out independent work in parallel. Faster, cheaper.
