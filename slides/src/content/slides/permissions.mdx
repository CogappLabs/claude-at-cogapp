## `/permissions` deep dive

Pre-approve safe, deny dangerous, leave the rest interactive.

```json
{
  "permissions": {
    "deny": [
      "Read(.env)",
      "Read(.env.*)",
      "Read(credentials.json)",
      "Read(~/.ssh/**)",
      "Bash(rm -rf:*)"
    ],
    "allow": [
      "Bash(npm test)",
      "Bash(npm run lint)",
      "Bash(git status)",
      "Bash(git diff:*)"
    ]
  }
}
```
