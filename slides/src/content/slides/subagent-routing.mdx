## Subagent routing

Don't let Opus do Haiku's job.

| Task | Agent | Why |
| :---- | :---- | :---- |
| File lookup, grep | Explore (Haiku) | Cheap, fast |
| Multi-file analysis | Sonnet | Right balance |
| Architecture, security review | Opus | Reasoning depth |

Tell Claude to pass `model: "haiku"` explicitly when spawning subagents (not inherit parent).
