## Skill scope

| Scope | Path | Available to |
| :---- | :---- | :---- |
| Personal | `~/.claude/skills/<name>/` | Your projects |
| Project | `.claude/skills/<name>/` | Anyone cloning repo |
| Enterprise | Managed settings | Whole org |

Be concise. Test with different models. Write descriptions that trigger well.
