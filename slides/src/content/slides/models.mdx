## Models

| Model | Best for | Speed | Cost |
| :---- | :---- | :---- | :---- |
| **Haiku** | Quick edits, file lookups, greps | Fast | Cheap |
| **Sonnet** | Day-to-day coding, reviews | Balanced | Mid |
| **Opus** | Architecture, complex bugs | Slower | Expensive |

Switch with `/model`. Plan with Opus, build with Sonnet.
