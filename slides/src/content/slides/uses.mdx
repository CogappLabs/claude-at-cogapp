## What to use it for

- **Debugging**: paste error, point at file
- **Self-review**: "review my staged changes for bugs"
- **Edge cases**: "what am I missing here?"
- **Tests**: "tests for UserProfile covering main flows"
- **Browser automation**: Chrome via Claude in Chrome
- **Git, refactors, boilerplate, explaining unfamiliar code**
