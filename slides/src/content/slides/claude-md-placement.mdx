## CLAUDE.md placement

| Path | Scope | Commit? |
| :---- | :---- | :---- |
| `~/.claude/CLAUDE.md` | Your machine, all projects | No |
| `<repo>/CLAUDE.md` | This project, everyone | Yes |
| `<repo>/.claude/CLAUDE.md` | This project, alt location | Yes |

Both repo locations work. Pick one convention per team.

### Client consent

Committing `CLAUDE.md` to a client repo = visible declaration AI is in use. Confirm client agreed to AI-assisted development first.

Options if not confirmed:
- Keep CLAUDE.md global (`~/.claude/`) only
- Or `.gitignore` the project CLAUDE.md (still loaded locally, not pushed)
