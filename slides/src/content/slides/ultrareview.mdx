## `/ultrareview`

Multi-agent cloud review of a branch or PR.

```
/ultrareview          # current branch
/ultrareview 1234     # PR number
```

Spawns parallel reviewers (security, perf, style, logic). Returns consolidated report. Billed separately.

User-triggered only. Claude can't launch it itself.
