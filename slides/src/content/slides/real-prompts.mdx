## Real prompts that work

**Craft / Twig:**
"Refactor this Twig include into a `_components/card.twig` partial."

**Drupal:**
"Explain this `hook_form_alter` chain in `mymodule.module`."

**Dagster / Polars:**
"Write a Pandera schema for this Parquet, then a transform asset."

**NTS:**
"Check Cloudflare logs for bot traffic on `/search` last 24h."

**Debugging:**
"Failing test `test_visitor_aggregation` — trace the data flow and find the regression vs main."

"This 500 hits prod once a day. Logs attached. Find the cause."

**Exploration:**
"Map the auth middleware chain in this repo. Diagram + file:line refs."

"Where does `User.export()` get called from, and what assumes the CSV format?"

**Prototyping:**
"Sketch a tiny Astro page showing daily TMS ingest counts as a line chart. No backend, fake data."

"Try three layouts for the visit-summary card. Side-by-side, no commits."
