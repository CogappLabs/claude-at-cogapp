## What is Claude Code?

Anthropic's AI coding assistant. Runs in terminal, desktop app, web, or IDE (VS Code, JetBrains) with direct access to files, git, shell.

- Read codebase, edit files, run commands
- Connects to external tools (DBs, browsers, Linear, Slack) via MCP
- Works across multiple files in one conversation
- 200k token context (1M on premium seats)
