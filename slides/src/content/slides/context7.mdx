## Context7

Up-to-date library docs at query time. No more hallucinated APIs.

```shell
npx ctx7 setup --claude
```

When prompted, choose **CLI + Skills** (not MCP server). Installs a skill that runs `ctx7` CLI commands to fetch docs.
