## Ways to use it

- **CLI**: run `claude` in a project dir
- **Desktop app**: Mac, Windows
- **Web**: claude.ai/code
- **IDE extensions**: VS Code, JetBrains *(less reliable with private repos and skill loading)*

Same Claude under the hood. Run `/init` to generate a starter `CLAUDE.md`.

CLI or desktop = best Cogapp experience.
