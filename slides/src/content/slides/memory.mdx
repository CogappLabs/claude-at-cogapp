## Memory system

> Cogapp convention layered on top of Claude Code's auto-memory feature. Vanilla Claude just has CLAUDE.md files.

Persistent file-based memory at `~/.claude/projects/<proj>/memory/`.

Four types:

- **user**: who you are, role, preferences
- **feedback**: corrections + confirmations
- **project**: current goals, deadlines, decisions
- **reference**: pointers to Linear, Grafana, etc.

Indexed via `MEMORY.md`. Survives across conversations.
