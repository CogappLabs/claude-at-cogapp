## Day-one setup checklist

1. Install Claude Code (CLI, desktop app, or IDE extension)
2. Sign in to Anthropic account
3. `cd` into a project, run `/init` for a `CLAUDE.md`
4. Drop in a global `~/.claude/CLAUDE.md` (your standards)
5. `npx ctx7 setup --claude` → CLI + Skills
6. Install Cogapp plugins: `cogapplabs/claude-plugins`
7. First test: ask it to summarise your branch
