## Trust ladder

How far can output go without your eyes?

```
Generated → Lint pass → Test pass → Local run → PR → Merge → Deploy
   ↑                                  ↑
   review here for risky              review here for routine
```

Generated code always reviewed before merge. Risky changes (migrations, auth, deletes) reviewed before running locally.
