## Anti-patterns

- Pasting full `.env` contents to "share context"
- Approving `Bash(*)` blanket
- Trusting generated migration SQL without review
- Letting it `git push --force` unsupervised
- Skipping pre-commit hooks because Claude said `--no-verify`
- Auto-approving every "delete this file" prompt
